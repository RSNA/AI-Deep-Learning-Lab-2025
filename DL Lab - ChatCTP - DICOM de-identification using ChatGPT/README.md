# Deep Learning Lab: ChatCTP - DICOM de-identification using ChatGPT

- Moderator: George Lee Shih MD MS
- Speaker: Adam Eugene Flanders MD
- Speaker: Errol Colak MD
- Speaker: Lisa Adams

## Overview

Hands-On learning lab that leverages conventional and AI tools (LLMs like ChatGPT / Gemini) to help with DICOM de-identification (DeID). Will practice DeID using scripts generated by LLMs and leverage GPT-4 in other ways for the de-identification process.  

1) Introduction to DICOM tags and PHI stored in these tags.
2) Explain the issues around DICOM de-identification (DeID) and provide examples of DeID tools.
3) Leverage Large Language Model (eg, Gemini Flash) to create a script to assist with DICOM DeID.
4) Hands-On DICOM DeID using GPT-4 generated scripts.
5) Other ways to leverage GPT-4 for DICOM DeID.
6) DICOM Pixel DeID
